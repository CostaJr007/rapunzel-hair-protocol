# 💇 Rapunzel Hair Protocol

Comprehensive hair care protocol and treatment tracker for healthy, long hair.

## 🎯 Features

- ✅ Daily care routine tracker
- ✅ Product inventory management
- ✅ Treatment schedule
- ✅ Progress photos
- ✅ Results analytics
- ✅ Product recommendations

## 📋 Requirements

- Web browser (for web version)
- Or Python 3.8+ (for CLI version)

## 🔧 Installation

### Web Version

Simply open index.html in your browser.

### CLI Version

 + "`" + @"
bash
git clone https://github.com/CostaJr007/protocolo-cabelo-rapunzel.git
pip install -r requirements.txt
 + "`" + @"

## 💡 Usage

### Daily Routine

1. Log your daily hair care activities
2. Track products used
3. Upload progress photos weekly
4. Review monthly analytics

### CLI

 + "`" + @"
bash
python main.py log --activity "mask" --product "Brand X"
python main.py progress --photo today.jpg
 + "`" + @"
